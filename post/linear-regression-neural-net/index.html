Fancy "Neural Network" Graphic but it's actually just Linear Regression

Linear regression but make it fancy

Michael DeCrescenzo
2020-07-21 2 min read Methods

Motivation

If you aren’t familiar with neural networks, you might have heard people (especially academics) joke about how “deep learning is just linear regression.” This might sound hard to believe, because after all, neural networks sound fancy, and they often introduced with a graphic like this…

which, despite being supposedly helpful, doesn’t really explain the intuition of what a neural network model really is.

The purpose of this post is to demystify neural networks by taking something easy—linear regression—and drawing it up in a “neural network-style graphic.” I very much intend to be cheeky about this: neural networks are not that complicated. If you know the algebra behind a generalized linear model, you know how to understand neural networks. The only reason this isn’t obvious is because neural networks have good marketing. But that’s what this post takes aim at.

GLMs and Neural Network Jargon

Neural network jargon contains terms like inputs, outputs, hidden nodes, hidden layers, weights, biases, activation functions…each of which has a pretty easy mapping to the basic structure of a GLM.

Let’s start with a linear model with Normal errors. For units indexed \(i\), the outcome \(y_{i}\) is a linear function of a vector of predictors \(\mathbf{x}_{i}\) and coefficients \(\beta\). We have \(P\) many predictors, so the length of \(\beta\) is also \(P\). Finally, we have a random Normal error \(\epsilon_{i}\)… \[\begin{align} \begin{split} y_{i} &= \alpha + \mathbf{x}_{i}^{\intercal}\beta + \epsilon_{i} \\ \epsilon_{i} &\sim \mathrm{Normal}\left(0, \sigma \right) \end{split} \tag{1} \end{align}\] We could write this more generally by construing \(y_{i}\) itself as a random variable with a mean that is conditional on data and parameters… \[\begin{align} \begin{split} y_{i} &\sim \mathrm{Normal}\left(\mu_{i}, \sigma \right) \\ \mu_{i} &= \alpha + \mathbf{x}_{i}^{\intercal}\beta \end{split} \tag{2} \end{align}\]

How does this relate to neural networks? Already we have almost every element of the neural network jargon represented. They go like so:

  • input: predictors \(\mathbf{x}_{i}\).
  • output: dependent variable \(y_{i}\).
  • weights: a vector coefficients, \(\beta\), length \(P\).
  • biases: a constant, \(\alpha\).
  • hidden node: a linear combination of input data \(\mathbf{x}_{i}\), weights \(\beta\), and “bias term” \(\alpha\). In regression this is equal to the linear predictor, \(\alpha + \mathbf{x}_{i}^{\intercal}\beta\).
  • activation function: a function the connects the linear predictor (or “hidden node”) to the dependent variable (“output”). This is similar to an inverse link function, although the neural network is slightly more general. In a linear regression example, the linear predictor is linked to the outcome by the identity function, so there is no substantive transformation of the linear predictor.
