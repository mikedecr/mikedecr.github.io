What Is the Mandelbrot Set?

The Mandelbrot Set is maybe the most recognizable fractal image out there. It looks like this in the macro:

…and like this up close.

We could zoom in on these images forever, if we had the computational power, and find the same patterns with infinitely repeating detail. It is an amazing thing.

These images are created from the math of the Mandelbrot Set, but the image isn’t really what the Mandelbrot Set is. Strictly speaking, the Mandelbrot Set is the set of complex numbers \(c\) that do not cause the function \[\begin{align} z_{t + 1} = z_{t}^2 + c \end{align}\] to explode to \(\pm\infty\) when iterated from the initial condition of \(z_{t} = 0\).

Uhh…

Remember complex numbers? Complex numbers contain some real component and some imaginary component, where the imaginary number \(i\) is equal to the square root of \(-1\) (and \(i^{2} = -1)\). You can think of real numbers as special cases of complex numbers, where the imaginary components are all zero. For example, the real number \(6\) is… \[\begin{align} 6 = 6 + 0i, \end{align}\] and the \(0i\) just drops out.

The images of the Mandelbrot Set are visualizations of complex numbers. Meaning, they are visualized in a coordinate system called the complex plane, which is a lot like the Cartesian plane—that is, \((x, y)\) coordinates—but where we have a real axis and an imaginary axis.
