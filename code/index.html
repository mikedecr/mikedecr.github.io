Sublime Text User Settings

Sublime Text is a powerful and customizable text editor that is great for social scientists. It has much of the extensibility as Vim and Emacs but with a friendlier interface. However, it does have text-based settings, which can be daunting for the uninitialized and a rabbit hole for the detail-obsessed.

I have some of my own custom settings available for pilfering on Github. Highlights include:

  • Keybindings (primarily for R and \(\mathrm{\LaTeX}\)) and associated macro files.
  • Code snippets for R and \(\mathrm{\LaTeX}\), plus some for Stan, Stata, HTML, and Markdown.
  • A modified Solarized color scheme based on the guts of the Cobalt2 scheme for Sublime, available in both light and dark versions.
